Appian Developer

The Appian Developer learning path is designed to teach participants how to build and configure an application that includes a variety of commonly-used design objects like record types, process models, interfaces, reports, and sites. This course series provides Developers with the tools and knowledge they need to deliver a functional and intuitive end-user experience that abides by best practices. Throughout the learning path, users should complete each course followed by hands-on, step-by-step exercises to build an application in their Appian Community Edition environment.

Appian Foundations

The Appian Foundations learning path is designed to introduce business users to Appian. Participants will learn more about integral application planning methods and recommendations, including the application development lifecycle and basic Agile concepts. We'll use Quick Apps to set the stage with regard to what a basic application might look like and show it from both the Designer and end user perspectives. Participants will also learn more about core application objects like interfaces and decision rules, which will help learners to get a sense for how application objects are configured. Lastly, participants will explore the world of user experience design in Appian and see how it can be used to dramatically impact how an application is perceived and used.
